HTML-Eigenschaften

Dokumentenstruktur
Tag Name Beschreibung
<!DOCTYPE> Document Type Definiert den Typ des HTML-Dokuments.
<html> HTML Das Wurzelelement eines HTML-Dokuments.
<head> Head Enthält Metadaten und Ressourcenverweise.
  <meta> Meta Definiert Metadaten wie Zeichencodierung.
  <title> Title Legt den Titel des Dokuments fest.
  <link> Link Verweist auf externe Ressourcen wie CSS.
<style> Style Enthält CSS-Regeln für das Dokument.
<script> Script Enthält JavaScript-Code oder Verweise.
<body> Body Enthält den sichtbaren Inhalt der Webseite.
Text Elemente
Tag Name Beschreibung
<h1> - <h6> Headings Überschriften von höchster bis niedrigster Ebene.
<p> Paragraph Absätze für Text.
<a> Anchor Hyperlinks zu anderen Seiten oder Ressourcen.
<img> Image Bilder.
<ul> Unordered List Ungeordnete Listen.
  <li> List Item Elemente in ungeordneten Listen.
<ol> Ordered List Geordnete Listen.
  <li> List Item Elemente in geordneten Listen.
<blockquote> Blockquote Blockzitate.
<q> Quotation Kurze Zitate im Text.
<em> Emphasis Betonter Text (kursiv).
<strong> Strong Stark hervorgehobener Text (fett).
<br> Line Break Erzwingt einen Zeilenumbruch.
<hr> Horizontal Rule Horizontale Trennlinie.
<code> Code Codeabschnitte.
<pre> Preformatted Vorformatierter Text.
Listen und Tabellen
Tag Name Beschreibung
<ul> Unordered List Ungeordnete Listen.
  <li> List Item Elemente in ungeordneten Listen.
<ol> Ordered List Geordnete Listen.
  <li> List Item Elemente in geordneten Listen.
<dl> Description List Beschreibungslisten.
  <dt> Definition Term Begriff in einer Beschreibungsliste.
  <dd> Definition Definition zu einem Begriff in einer Liste.
<table> Table Tabellen zur Anordnung von Daten.
  <caption> Table Caption Titel oder Beschreibung für die Tabelle.
  <thead> Table Head Abschnitt für Überschriften der Tabelle.
  <tbody> Table Body Abschnitt für Hauptdaten der Tabelle.
  <tr> Table Row Zeilen in Tabellen.
    <th> Table Header Tabellenkopfzellen.
    <td> Table Data Tabellendatenzellen.
  <tfoot> Table Foot Fußzeile der Tabelle.
